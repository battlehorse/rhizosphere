Contribute to Rhizosphere development

Rhizosphere is an open source library. We welcome all kind of contributions, from small fixes in the documentation to code patches, to integrations of Rhizosphere with other applications and datasources.

If you want to contribute or follow Rhizosphere development:

  • Follow Rhizosphere announcements either on via Twitter or the Announcements blog .
  • Join the Rhizosphere discussion group and let us know your opinions.
  • File bugs or feature requests in the Rhizosphere Issue tracker, or scan the list of open issues to find something you might want to work on.
  • Clone the Rhizosphere source code, try your ideas out and submit them back for inclusion in the library.

About submitting patches and code changes

If you modified Rhizosphere source code and you want your changes to be included in the library, send us a patch.

For small changes, just use the Patch submission form and attach the patch directly to it.

For bigger changes, it would be great if you could first roll out your patch in a cloned Rhizosphere repository and then notify us via the Patch submission form, so we can do a proper code review on your submission if necessary.

For any questions, feel free to contact the team via the discussion group.
