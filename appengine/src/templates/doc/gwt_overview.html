Google Web Toolkit integration

If you write web applications using Google Web Toolkit, you can use Rhizosphere within them thanks to the GWT-Rhizosphere integration library. This section describes how to integrate Rhizosphere with your GWT app.

Overview


(A screenshot from the GWT demo application, Click to enlarge).

Rhizosphere is available within GWT as a GWT module you can embed in your application. It works by providing a Java facade to the underlying Rhizosphere javascript library which will be embedded in your application.

Rhizosphere-GWT gives you access to the entire set of Rhizosphere features via a Java API that hides all the javascript internals. There is no need to write JSNI code, with the exception of a few advanced scenarios. Additionally, it lets you use all the Java and GWT idioms you are already familiar with, such as using Java annotations for configuration, the GWT widget library and UiBinder templates for UI definition.

Rhizosphere pays close attention to performance: it is easy to isolate Rhizosphere so that its libraries will only be loaded when effectively needed by the user, therefore not impacting your application load times and responsiveness.

Get Started

Rhizosphere-GWT is built on top of the javascript Rhizosphere library. Most of the concepts that are valid for the js library are equally valid for the GWT version. Make yourself familiar with the rest of the documentation and especially with the library Concepts.

In addition to the documentation, the following resources are available to help using Rhizosphere in your GWT apps:

  • A GWT demo application that showcases various forms of integration and library features,
  • The source code for both the demo application and the entire Rhizosphere-GWT module,
  • javadocs covering the Rhizosphere-GWT module.

Adding Rhizosphere to your GWT application is as simple as downloading gwt-rhizosphere.jar from the Rhizosphere donwloads page and including it in your application classpath. Alternatively you can build it from source following the instructions in the GWT Development Setup page.
