Features

Rhizosphere project focuses both on traditional and innovative ways to browse through data. Here follows a selection of library features.

Rhizosphere launches and iterates often. Keep an eye at the Announcements section of the project site, or follow Rhizosphere on Twitter for the latest news and releases.

Pure Javascript and HTML5
No Flash. No Silverlight. No plugins needed. Your browser alone is just fine. We leverage HTML5 features to provide a better user experience (see this example , about history and state management).
Small footprint
The Rhizosphere library is only 26Kb once minified and gzipped. Once you include the dependent libraries (jQuery, jQuery UI) it fits within 60-70Kb. Client side caching means the user pays the load cost only the first time it uses the library.
Speed
Sub-second response times for most operations, even on big datasets with thousands of datapoints. See latest benchmarks.
From high end PCs to Portable devices
From high-power desktop PCs all the way down to tablets and smartphones. You can run Rhizosphere visualizations on all of them.
Embeddable
You can use Rhizosphere to create a stand-alone full-page visualization, or you can embed it anywhere you want: in your webpages, as a Google Spreadsheet or iGoogle gadget. You can have multiple visualizations per page.
Google Web Toolkit integration
Rhizosphere fully integrates with Google Web Toolkit. If you develop applications with GWT, you can use Rhizosphere inside your apps without having to write a single line of Javascript, and taking advantage of GWT features you are used to, like UiBinder templates and the widget framework.
Spatial navigation
Rhizosphere behaves like a physical desktop with support for drag'n'drop, groups selections, zooming, endless panning (aka Google Maps-like scrolling)
Multiple layout algorithms
To help you organize your data: plain, clusters, trees, treemaps and more.
Dynamic filters
To let you isolate, slice, partition your datasets and focus only on the subset of data you care about.
UI Customization
You can tweak any aspect of the visualization, from the color scheme to use to the UI layout, up to the rendering algorithm of each data point you visualize.
Integration with multiple data sources
You can feed Rhizosphere data via Google Visualization API datasources, Google Spreadsheets or using its custom input format. Custom data sources can be added easily.
