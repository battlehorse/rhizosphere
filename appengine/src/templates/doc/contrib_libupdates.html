Third party library updates

This page lists the procedures to follow when updating third-party libraries Rhizosphere depends upon.

jQuery UI

Rhizosphere uses both a custom theme and packaging of jQuery UI components. To update the version of jQuery UI used, for each visual theme Rhizosphere supports:

  • light: corresponding jQuery UI theme,
  • default: corresponding jQuery UI theme.

Perform the following series of steps:

  • Download the corresponding jQuery UI theme for each Rhizosphere theme by navigating to each of the above links,
  • Use the custom .rhizo CSS scope,
  • Include the following packages when building the jQuery UI download: draggable, droppable, selectable, slider, progressbar.
  • From the downloaded bundle copy the following files into the Rhizosphere source tree: jquery-ui-1.8.16.custom.min.js, jquery-ui-1.8.16.custom.js, jquery-ui-1.8.16.custom.css and all the images/ associated with the custom style. The css and images must positioned inside the corresponding theme directory.

Note. You can always find the most recent jQuery UI themeroller urls in the generated jQuery UI CSS files, included in Rhizosphere source tree.
