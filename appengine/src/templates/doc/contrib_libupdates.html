Third party library updates

This page lists the procedures to follow when updating third-party libraries Rhizosphere depends upon.

jQuery UI

Rhizosphere uses both a custom theme and packaging for jQuery UI components. To update the version of jQuery UI used:

  • Use this custom themeroller theme (you can always find the most recent jQuery UI themeroller url in the generated jQuery UI CSS file).
  • Use the custom .rhizo CSS scope.
  • Include the following packages when building the jQuery UI download: draggable, droppable, selectable, slider, progressbar.
  • From the downloaded bundle copy the following files into the Rhizosphere source tree: jquery-ui-1.8.16.custom.min.js, jquery-ui-1.8.16.custom.js, jquery-ui-1.8.16.custom.css and all the images/ associated with the custom style.
