Requirements

Rhizosphere is 100% web-based, and only uses standard technologies like HTML and Javascript. You only need your browser to access Rhizosphere visualizations.

Rhizosphere leverages HTML5 features which may not be available to all current browsers, but it always provides fallback functionality for missing browser capabilities.

Supported browsers

Rhizosphere supports the following browsers:

Google Chrome
This is the recommended choice for speed and performance.
Apple Safari
Tested against version 5. Should work on version 4 and 3 too.
Mozilla Firefox
Tested against version 3.6 and (upcoming) version 4. Should work with earliers versions (FF3, possibly FF2).
Opera
Tested againg version 10.60 and onward.

Note. Microsoft Internet Explorer is not actively supported, although we tested newly released version 9 and is likely to work flawlessly. For earlier versions we recommend installing Google Chrome Frame.

Supported devices

Rhizosphere has been tested on traditional laptops, netbooks and desktop PCs.

Rhizosphere has been tested on a range of iOS devices: Apple iPad, iPhone and iPod Touch. On smartphones and tables it provides a customized user experience that adapts to the device screen size.

Note. Rhizosphere also provides experimental support for Android devices.
