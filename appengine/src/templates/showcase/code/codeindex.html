Google Code Issue Tracking

Powered by the Rhizosphere visualization library. Learn more

Rhizosphere is an innovative project to provide in-browser html-based spatial navigation and interaction on structured data.

This showcase application integrates Rhizosphere with Google Code Project Hosting to let you browse and manage issues, tasks and activities associated to your Google Code projects.

This application exists to demonstrate Rhizosphere capabilities. Although we aim to provide a useful service, we may not be able to offer a rich enough feature set to cover all project management needs.

If you want to learn more, you can browse the Rhizosphere site, have a look at the Rhizosphere source code (it's open source), read the documentation or ask questions in the discussion group.
